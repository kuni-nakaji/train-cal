# たしざんれんしゅう！

子ども向けのたしざん練習アプリです。ブラウザ上でAIが手書き数字を認識し、1〜10までの足し算問題を5問出題します。

## 特徴

- **手書き認識** — TensorFlow.js（CNN）でブラウザ内学習・推論。サーバー不要。
- **「10」対応** — 「1」と「0」に分割して認識するジオメトリックフォールバック実装。
- **Apple Pencil対応** — Pointer Events APIで筆圧に応じた線幅を実現。
- **十字グリッド** — 数字を書く位置の目安になる点線ガイドをキャンバスに表示。
- **Claymorphismデザイン** — 子ども向けの丸みのあるカード、スプリングアニメーション、shimmerプログレスバー。
- **iPad最適化** — バッチレベルのUI yield処理で学習フリーズを防止。

## ファイル構成

```
index.html   — HTMLマークアップ
style.css    — CSSスタイル（Claymorphism、アニメーション、レスポンシブ）
app.js       — JavaScript（ML学習・推論、ゲームロジック、キャンバス描画）
```

## 使い方

静的ファイルをサーブするだけで動きます。

```bash
npx serve .
# または
python3 -m http.server
```

初回アクセス時にブラウザ内でAIが学習します（約10〜30秒）。学習済みモデルはIndexedDBにキャッシュされるため、2回目以降は即座に起動します。

## 技術スタック

| 項目 | 内容 |
|------|------|
| ML フレームワーク | TensorFlow.js 4.17.0 |
| モデル | CNN（Conv2D × 2 + Dense 32 + Dropout） |
| 学習データ | ブラウザ内でストロークベースの手書き数字を生成（クラスあたり70〜120サンプル） |
| フォント | Baloo 2 / Nunito（Google Fonts） |
| モデルキャッシュ | IndexedDB |
| ターゲット端末 | iPad / スマートフォン / デスクトップ |
